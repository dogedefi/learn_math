欧几里得定理 —— 几何原本，四定理：
    1.两点定一线
    2.线段延伸还是直线
    3.圆心+半径构成圆
    4.直角相等
    5.直线外一点有且只有一条平行线（不成立）
    // 以上皆为欧几里得几何
    5´.直线外一点有多条平行线（罗巴切夫，罗氏几何）（空间是非曲面的）
    5˝.直线外一点没有条平行线（黎曼，黎曼几何）（空间是曲面的）
    // 非欧几里得几何 
    
泰勒斯定理 —— 4个几何定理（非曲面空间）。看了典故，大佬是利用了同构的思维，先从自身的影长变化，得出公式，进而推算金字塔的高度
1.圆的直径将圆分为两个相等的部分;
2.等腰三角形两底角相等;
3.两相交直线形成的对顶角相等;
4.如果一三角形有两角、一边分别与另一三角形的对应边，对应角相等，那么这两个三角形全等.
